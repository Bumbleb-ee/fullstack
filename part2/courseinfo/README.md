# Part 2: Course Information Exercises 2.1-2.5

This project contains solutions for Fullstack Open Part 2, exercises 2.1-2.5.

## Structure
- All code is in the `src` folder.
- The main component is `App.jsx`.
- The `Course` component and its subcomponents are in `Course.jsx`.

## How to run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Exercises
- 2.1: Render course contents using components
- 2.2: Show sum of exercises
- 2.3: Use `reduce` to calculate sum
- 2.4: Support arbitrary number of courses
- 2.5: Course component in a separate module
